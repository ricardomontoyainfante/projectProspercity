Introducción al HTML

HTML (Hyper Text Markup Language) es el lenguaje de marcado estándar para páginas Web. Con HTML puedes crear tu propio sitio web!.

El leguaje HTML tiene una estructura básica en donde se introducen todas las etiquetas con las que se contruyen las páginas web que conocemos.

<!DOCTYPE html> <html>
<head>
<title>Título de la página</title>
</head>
<body>
<h1>Mi primer encabezado</h1>
<p>Mi primer parrafo</p>
</body>
</html>

Ejemplo explicado

  • Esta etiqueta define que este documento es de tipo HTML5 <!DOCTYPE html>
  • El elemento es la etiqueta raíz de una página HTML <html>
  • El elemento contiene meta-información sobre la página HTML <head>
  • El elemento especifica el título para la página HTML (que se muestra en la barra de título del navegador o en la pestaña de la página) <title>
  • El elemento define el cuerpo del documento, y es un contenedor para todos los contenidos visibles, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc. <body>
  • El elemento define un encabezado grande <h1>
  • El elemento define un párrafo <p>

Elementos HTML

Los elementos HTML se definen mediante una etiqueta de inicio, algo de contenido y una etiqueta de fin:

<tagname> El contenido va aquí... </tagname>

El elemento HTML es todo, desde la etiqueta de inicio hasta la etiqueta de fin:

<h1>Mi primer encabezado</h1>
<p> Mi primer párrafo. </p>

¿En que programa puedo construir páginas web?

Puedes crear paginas web utilizando diferentes programas, si eres principiante te recomiendo empezar usando el blog del notas o Notepad. Puedes usar también programas como Brackets, Dreamweaver, etc; estos últimos con herramientas que facilitarán el proceso de programación.
