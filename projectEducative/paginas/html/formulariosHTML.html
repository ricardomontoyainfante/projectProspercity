Formularios HTML

Los formularios son utilizados para capturar información del usuario. El elemento se crea utilizando la etiqueta <form>

<form>

   Elementos de formulario

</form>

El elemento es un contenedor para diferentes tipos de elementos de entrada, tales como:

  • Campos de texto
  • Casillas de verificación
  • Botones de opción
  • Botones de envío

Los elementos de un formulario se pueden encontrar en el siguiente enlace: https://www.w3schools.com/html/html_form_elements.asp. A continuación se presentan los elementos mas utilizados en los formularios HTML:

Elemento input <input>

Es el elemento mas utilizado, se puede mostrar de muchas maneras, modificando el atributo "type" <input type=" ">, estos son algunos de ellos:

Tipo Descripción Apariencia
<input type="text"> Muestra un campo de entrada de texto de una sola línea
<input type="radio"> Muestra un botón de opción (para seleccionar una de las muchas opciones)
<input type="checkbox"> Muestra una casilla de verificación (para seleccionar cero o más de muchas opciones)
<input type="submit"> Muestra un botón de envío (para enviar el formulario)
<input type="button"> Muestra un botón en el que se puede hacer click.

En el siguiente enlace encontrarán todos los tipos (type de input: https://www.w3schools.com/html/html_form_input_types.asp

Elemento label <label>

Este elemento define una etiqueta para muchos elementos de formulario. El elemento es útil para los usuarios de lectores de pantalla, porque el lector de pantalla leerá en voz alta la etiqueta cuando el usuario se centre en el elemento.

El elemento también ayuda a los usuarios que tienen dificultades para hacer clic en regiones muy pequeñas (como botones de opción o casillas de verificación), porque cuando el usuario hace clic en el texto dentro del elemento, alterna el botón de opción / casilla de verificación.

El atributo for de la etiqueta <label> debe ser igual al atributo id del elemento <input> para unirlos.

A continuación se presenta la forma de enlazar los elementos <label> con los elementos del <form>.

Atributos de formularios <form>

Atributo action <form action="">

El atributo action define la acción que se va a realizar cuando se envía el formulario. En el siguiente ejemplo, los datos del formulario se envían a un archivo llamado "action_page.php". Este archivo contiene un script del lado del servidor que controla los datos del formulario:

<form action="/action_page.php"> <label for="fname">First name:</label><br>
<input type="text" id="fname" name="fname" value="John"><br>
<label for="lname">Last name:</label><br>
<input type="text" id="lname" name="lname" value="Doe"><br><br>
<input type="submit" value="Submit"< </form>





Atributo method <form method="">

El atributo especifica el método HTTP que se utilizará al enviar los datos del formulario.

Los datos del formulario se pueden enviar como variables URL (get) o como transacción HTTP post (post). method="get"method="post"

El método HTTP predeterminado al enviar datos de formulario es GET.

Notas sobre GET:

  • Anexa los datos del formulario a la dirección URL, en pares nombre/valor
  • ¡NUNCA use GET para enviar datos confidenciales! (¡los datos del formulario enviados son visibles en la URL!)
  • La longitud de una URL es limitada (2048 caracteres) Útil para envíos de formularios en los que un usuario desea marcar el resultado
  • GET es bueno para datos no seguros, como cadenas de consulta en Google

Notas sobre POST:

  • Anexa los datos del formulario dentro del cuerpo de la solicitud HTTP (los datos del formulario enviados no se muestran en la dirección URL)
  • POST no tiene limitaciones de tamaño y se puede utilizar para enviar grandes cantidades de datos. Los envíos de formularios con POST no se pueden marcar como favoritos

Atributo autocompletar <form autocomplete="">

El atributo especifica si un formulario debe tener autocompletado activado (on) o desactivado (off).

Cuando el autocompletado está activado, el navegador completa automáticamente los valores en función de los valores que el usuario ha introducido antes.
